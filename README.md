# PACH NW At Home Oasis

Standalone marketing site for PACH NW At Home Oasis.

## Notes

- This project is intentionally independent from any TieGui folders or repositories.
- Visual direction is permaculture-focused and uses original copy, styling, and placeholder artwork.
- The contact form now uses the API route at `app/api/contact/route.ts`.
- Add the SMTP variables from `.env.example` in Vercel before relying on the live form.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
- `npm run check-types`
